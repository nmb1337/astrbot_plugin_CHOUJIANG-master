# astrbot_plugin_choujiang

AstrBot 群抽奖插件，支持：

- 报名抽奖
- 自定义开奖时间
- 自定义奖品
- 到点自动开奖并 @全体
- 自定义提醒时间
- 提醒时 @未报名成员（OneBot v11/aiocqhttp）

## 使用前提

- AstrBot >= 4.9.2（使用插件 KV 存储）
- 推荐平台：aiocqhttp（支持获取群成员并逐个提醒）

## 指令说明

- `/抽奖帮助`
- `/抽奖创建 <开奖时间> | <奖品>`
	- 示例：`/抽奖创建 2026-05-03 20:00 | 京东卡100元`
- `/抽奖报名`
- `/抽奖名单`
- `/抽奖奖品 <新奖品>`
- `/抽奖开奖时间 <时间>`
	- 示例：`/抽奖开奖时间 2026-05-03 20:30`
- `/抽奖提醒时间 <时间>`
	- 支持绝对时间：`2026-05-03 19:50`
	- 支持相对时间：`30m`、`2h`、`45s`
- `/抽奖提醒前 <时长>`
	- 示例：`/抽奖提醒前 30m`（开奖前 30 分钟提醒）
- `/抽奖开奖`（立即开奖）
- `/抽奖取消`

## 时间格式

支持以下格式：

- `YYYY-MM-DD HH:MM:SS`
- `YYYY-MM-DD HH:MM`
- `YYYY/MM/DD HH:MM:SS`
- `YYYY/MM/DD HH:MM`
- 相对时长：`30m`、`2h`、`45s`、`1d`

## 行为说明

- 每个群同一时间只保留 1 个进行中的抽奖。
- 到开奖时间后，插件会自动开奖并公布中奖者。
- 自动开奖时会发送 `@全体`。
- 到提醒时间时，插件会优先尝试拉取群成员列表并 `@未报名成员`；若平台不支持，则回退为 `@全体` 提醒。
